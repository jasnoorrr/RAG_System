Project: RAG Robustness Against Distractors

How to run:
1. Install dependencies:
   pip install numpy pandas matplotlib scikit-learn sentence-transformers rank-bm25

2. Place dataset:
   News_Category_Dataset_v3.json in same folder

3. Run:
   python ragsystem.ipynb

Description:
This project evaluates retrieval robustness using Dense, BM25, TF-IDF, Hybrid, Defended Dense, and Logistic Regression Ranker.
